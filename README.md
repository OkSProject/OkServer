# OkServer!
_OkServer!_, or _OkS!_ for short, is a Go-based HTTP web server aimed to be lightweight, clean, and straightforward.

## Installation
1. Run ```git clone https://github.com/OkSProject/OkServer.git```
2. Run ```cd OkServer```
3. Run ```./okserver``` or ```./server.go```
